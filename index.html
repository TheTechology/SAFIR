Caracteristici

Tehnologie de Ultimă Generație

SAFIR integrează AI, senzori IoT și educație comunitară pentru irigații durabile și adaptare climatică.

🔍

Prognoză Apă Subterană

Modele avansate de previziune pentru nivelul apei subterane bazate pe senzori în timp real și date istorice.

🧠

LSTM Umiditate Sol

Rețea neurală LSTM care previzionează umiditatea solului pentru optimizare automată a irigațiilor.

☀️

Control Solar

Logică inteligentă pentru controlul irigațiilor pe bază de energie solară disponibilă și stare baterie.

🤖

Învățare prin Consolidare

Algoritmi RL pentru planificarea adaptivă și optimizarea deciziilor de irigație în timp real.

📡

API Deschis

Endpoint-uri REST complete pentru acces la date, prognoze și configurații. Documentație Swagger inclusă.

📊

Date Publice

Seturi de date ale senzorilor disponibile sub licență MIT. Transparență și reproducibilitate totală.

De la Idee la Implementare

Parcursul SAFIR — de la conceptualizare la lansarea publică internațională.

Sept 2025
Conceptualizarea și Propunerea
Asociația Grupul Verde identifică nevoia de monitorizare inteligentă a irigațiilor în regiunile vulnerabile la secetă.
Oct – Dec 2025
Formarea Echipei și Cercetare
55 de tineri din 5 țări europene se alătură proiectului. Cercetare date climatice, senzori IoT și arhitectura AI.
Ian – Mar 2026
Dezvoltarea Modelelor AI
Implementarea LSTM, reinforcement learning și prognoze apă subterană. Prime teste pe date sintetice.
Apr – Mai 2026
Pilot în Laos
Validarea senzorilor pe teren, calibrarea modelelor pe date reale și integrarea cu sistemele locale.
Iun 2026
Lansare Publică & Open Source
SAFIR disponibil public sub licență MIT. API documentat, date și cod sursă publicate internațional.

API & Documentație

Integrare Simplă, Putere Reală

API REST complet pentru integrare și preluare date. Disponibil la /docs pe serverul de aplicație.

Endpoint-uri Principale
POST/sensor — Trimite date senzor
GET/sensor — Obține toate datele
GET/dashboard — Sumar tablou de bord
GET/forecast/groundwater — Prognoză apă
GET/project/info — Informații proiect
GET/project/team — Info echipă
GET/api/status — Status API
⚙ Instalare și Lansare
pip install -e . safir serve --host 0.0.0.0 --port 8000 # Documentație API: # http://localhost:8000/docs
